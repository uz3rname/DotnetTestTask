# .NET test task

## Использованные технологии:

- .NET Core v8
- PostgreSQL v15.4
- Entity Framework
- Blazor

## Запуск

Для запуска проекта выполните:

```
git clone https://github.com/uz3rname/DotnetTestTask
cd DotnetTestTask
docker compose up
```

Требуется установленный Docker.
Web интерфейс будет доступен по адресу http://localhost:8080/
